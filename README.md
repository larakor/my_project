# "Проект my_project"

## Описание:

**Этот Проект - ученический. 
Это виджет для ЛК клиента банка, показывает несколько послених успешных операций клиента**

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/larakor/my_project.git
```

2. Установите зависимости:
```
pip install -r requirements.txt
```

## Использование:

1. Модули masks, widget и processing, которые содержат функции, обрабатывающие информацию о картах и счетах.
2. В папке tests созданы файлы с тестами для каждого модуля в папке src соответственно.
3. Модуль generators содержит функции, реализующие генераторы для обработки данных, все функции тестируются в файле 
test_generators
